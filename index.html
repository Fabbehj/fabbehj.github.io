HTML Tabell-referenser

Tabelltaggar

Tag Namn Beskrivning
<table> Table Omsluter alla tabell-element
<thead> Table Head Raderna som definierar kolumnernas huvud
<tbody> Table Body Raderna som inehåller all data
<tr> Table Row Tabellens rader
<td> Table Data Datan i raderna
<tfoot> Table Foot Raderna som är tabellens fot

Tabellattribut

Attribut Namn Beskrivning
colspan Column Span Definierar hur många kolumner som ett td element ska övergripa
rowspan Row Span Definierar hur många rader som ett td element ska övergripa
